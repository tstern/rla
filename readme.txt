1. Projekt von https://github.com/tstern/rla klonen.
2. Im Client Verzeichnis die Befehl "npm install" und "bower install" ausführen.
3. Im Server Verzeichnis den Befehl "npm install" ausführen und einen Ordner "public" anlegen.
4. Im Client Verzeichnis den Befehl "grunt build" ausführen und die generierten Dateien im "dist" Ordner kopieren.
5. Die kopierten Dateien in den "public" Ordner einfügen.
6. Im Server Verzeichnis den Befehl "node server.js" ausführen.
7. Im Browser "localhost:8080" aufrufen.
